# <a name="powerapps-community-plan-a-free-development-environment-for-individual-use"></a>Fellesskapsplan for PowerApps: et gratis utviklingsmiljø for individuell bruk
Hvis du vil utvikle ferdigheter og lære mer om PowerApps, Microsoft Flow og Common Data Service, er fellesskapsplanen for PowerApps den rette planen for deg. Fellesskapsplanen for PowerApps gir deg et gratis utviklingsmiljø for individuell bruk der du kan:

* Lære hvordan du lager forretningsapper og arbeidsflyter med alle funksjonene i PowerApps og Microsoft Flow.
* Koble til en datakilde ved hjelp av våre 100+ [medfølgende koblinger](./canvas-apps/connections-list.md) eller ved å opprette [egendefinerte koblinger](./canvas-apps/register-custom-api.md).
* Finne ut hvordan du kan bruke [Common Data Service](https://docs.microsoft.com/common-data-service/entity-reference/introduction) til å bygge kraftige forretningsapper med Common Data Model og [SDK](https://aka.ms/eek20s).
* Eksportere løsningene du oppretter i det individuelle-miljøet, og [oppføre dem på AppSource](./canvas-apps/dev-appsource-test-drive.md) slik at kunder kan teste dem.

## <a name="who-can-sign-up-for-the-powerapps-community-plan"></a>Hvem kan registrere seg for fellesskapsplanen for PowerApps?
Alle som har en [arbeids- eller skolekonto](signup-for-powerapps.md#faq), kan registrere seg for fellesskapsplanen for PowerApps. Men vi anbefaler spesielt denne planen hvis du:

* Vil utvikle ferdigheter og finne ut mer om PowerApps, Microsoft Flow og Common Data Service.
* Er interessert i å bygge forretningsapper og arbeidsflyter for distribusjon på AppSource.

## <a name="where-can-i-sign-up-for-the-powerapps-community-plan"></a>Hvor kan registrere meg for fellesskapsplanen for PowerApps?
Registrer deg på [nettstedet for PowerApps-fellesskapsplan](https://powerapps.microsoft.com/communityplan). Hvis du er en eksisterende bruker PowerApps med Office 365 eller Dynamics 365, kan du også [opprette et miljø for individuell bruk](https://web.powerapps.com/community/signup).

Når du har registrert deg for fellesskapsplanen, vil du bli omdirigert til [PowerApps-nettstedet](https://web.powerapps.com?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) og kommer til det indviduelle miljøet. Miljøet bærer navnet ditt, for eksempel "Ola Nordmann-miljøet". Hvis det allerede finnes et miljø med samme navn, gis det individuelle miljøet navnet "Ola Nordmann (1)-miljøet".  Illustrasjonen nedenfor viser hvordan miljøet vises.

![Individuelt miljø for fellesskapsplan](./media/dev-community-plan/individual-environment.png)

### <a name="get-the-community-plan-with-visual-studio-dev-essentials"></a>Få fellesskapsplanen med Visual Studio Dev Essentials
Hvis du bruker Visual Studio Dev Essentials, er PowerApps inkludert i fordelene. Gå til [Mine fordeler](https://my.visualstudio.com/benefits) og klikk eller trykk på PowerApps-flisen for å registrere deg for PowerApps-fellesskapsplanen.

![Fellesskapsplan i Visual Studio](./media/dev-community-plan/visual-studio.png)

## <a name="which-features-are-included-in-the-powerapps-community-plan"></a>Hvilke funksjoner er inkludert i fellesskapsplanen for PowerApps?
Du får følgende funksjonalitet med det individuelle miljøet:

| **Funksjoner** | **Miljø for individuell bruk** |
| --- | --- |
| **Viktige funksjoner** | |
| Opprette og kjøre apper |Ja. Du kan opprette ubegrensede apper |
| Dele apper\* |Nei |
| Bruke Common Data Service |Ja** |
| Modellere dataene ved hjelp av Common Data Service |Ja** |
| Administrasjon på bedriftsnivå av miljøet og brukerpolicyer |Ja |
| **Tilkobling** | |
| Koble til Office 365, Dynamics 365 og andre koblinger |Ja |
| Koble til skybaserte tjenester, for eksempel Azure SQL, Dropbox, Twitter og mange flere |Ja |
| Bruke premium-koblinger som Salesforce, DB2 og mange flere |Ja |
| Tilgang til lokale data ved hjelp av en lokal gateway |Ja |
| Opprette egendefinerte koblinger for å koble til dine egne systemer |Ja. Du kan opprette et ubegrenset antall egendefinerte koblinger |
| **Common Data Service** | |
| Opprette og kjøre programmer på Common Data Service |Ja** |
| Modellere dataene i Common Data Service |Ja** |
| Opprette en database i Common Data Service |Ja** |
| **Behandling** | |
| Legge til kolleger som miljøopprettere og administratorer |Nei |
| Legge til kolleger i databaserollene |Nei |
| Støtter datapolicyer etablert av Office 365-administrator |Ja |
| Opprette datapolicyer for det individuelle miljøet |Ja |

*Du kan ikke dele apper, flyter, tilkoblinger osv. med andre brukere av leieren din. Du kan heller ikke legge til en annen bruker som en miljøadministrator eller -oppretter, eller til databaserollene fra administrasjonssenteret.

\*\*Oppretting av en database med Common Data Service er for øyeblikket ikke tilgjengelig i et individuelt miljø, men blir snart tilgjengelig igjen. Hvis du har opprettet en database tidligere, kan du fortsette å bruke den uten problemer. 

## <a name="what-are-the-capacity-limits-for-the-individual-environment"></a>Hva er kapasitetsgrensene for det individuelle miljøet?

| **Kapasitet** |  |
| --- | --- |
| Flytkjøringer per måned |750 |
| Databasestørrelse |200 MB |
| Fillagring |2 GB |

Du kan ikke bruke tillegg i antallet vi inkluderer. Hvis det oppstår kapasitetsgrenser, anbefaler vi at du kjøper PowerApps Plan 2. Lær mer om den fra [siden med priser for PowerApps](https://powerapps.microsoft.com/pricing/).

> [!NOTE]
> Kapasiteten for det individuelle miljøet, enten det brukes eller ikke, bidrar ikke til firmaets totale salgsbudsjett.

## <a name="publishing-to-appsource"></a>Publisering til AppSource
Har du en app som du ønsker å dele med kunder? Vi støtter nå en testløsning for PowerApps på [AppSource.com](https://appsource.microsoft.com) som en måte du kan dele apper og flyter med kunder på, og generere kundeemner for firmaet. Hvis du vil ha mer informasjon, kan du se [La kunder teste appene dine på AppSource](./canvas-apps/dev-appsource-test-drive.md).

## <a name="frequently-asked-questions"></a>Vanlige spørsmål
**Sp.** Hva gjør jeg hvis jeg når kapasitetsgrensene for miljøet?

**Sv:** Kapasiteten er begrenset fordi dette miljøet er ment for individuell bruk, ikke for et team eller produksjonsbruk. Kapasitet som leveres er:

| **Kapasitet** |  |
| --- | --- |
| Flytkjøringer per måned |750 |
| Databasestørrelse |200 MB |
| Fillagring |2 GB |

Hvis du når én eller flere kapasitetsgrenser, anbefaler vi at du har kjøper en plan som støtter produksjonsbruk. Lær mer om våre planer og deres begrensninger på [siden med priser for PowerApps](https://powerapps.microsoft.com/pricing/).

**Sp.** Kan jeg overføre appene, flytene og andre ressurser som er opprettet i det individuelle miljøet, til et annet miljø?

**Sv:** Ja, du skal kunne eksportere ressursene fra dette miljøet til andre miljøer. For å få mer informasjon, se [Overføring av miljø og leierapp](../administrator/environment-and-tenant-migration.md).

**Sp.** Vil abonnementet mitt på fellesskapsplanen for PowerApps noen gang utløpe?

**Sv:** Du kan bruke abonnementet ditt på fellesskapsplanen for PowerApps gratis så lenge du vil. Hvis du bruker et enkelt miljø aktivt, mister du ikke tilgangen til noen av ressursene eller funksjonene i miljøet. Du kan imidlertid oppleve en forsinkelse første gang du åpner Common Data Service-databasen etter en lang periode uten aktivitet.  Denne forsinkelsen påvirker ikke dataene eller enhetene som lagres i Common Data Service.

**Sp.** Kan jeg få eller opprette flere individuelle miljøer?

**Sv:** Nei, du kan bare ha ett enkelt miljø som opprettes av PowerApps når du registrerer deg for fellesskapsplanen.

**Sp.** Hva er forskjellen mellom PowerApps Plan 2-prøveversjonen og fellesskapsplanen for PowerApps, og hvilken bør jeg registrere meg for?

**Sv:** Begge er gratis, men de er opprettet for ulike formål:

**PowerApps Plan 2-prøveversjon** gir deg PowerApps Plan 2 i 30 dager. Dette er ment for testing av PowerApps, Common Data Service og Microsoft Flow. Når prøveversjonen utløper, kan du kjøpe en plan. Hvis du allerede bruker PowerApps med Office 365 eller Dynamics 365, er dette den rette planen for å teste premium-funksjonaliteten til PowerApps, som er tilgjengelig med PowerApps Plan 2.

**Fellesskapsplanen for PowerApps** gir deg tilgang til PowerApps premium-funksjonalitet, Common Data Service og Microsoft Flow for individuell bruk. Denne planen er først og fremst ment for læring eller oppretting av forretningsløsninger som skal distribueres [AppSource-test](./canvas-apps/dev-appsource-test-drive.md). Denne planen er tilgjengelig for alltid, men du kan bare bruke den til å lære om å bygge opp dine ferdigheter på PowerApps, Common Data Services og Microsoft Flow.

**Sp.** Kan jeg registrere meg med den personlige kontoen min?

**Sv:** Nei, du kan bare registrere deg med [arbeids- eller skolekontoen din](signup-for-powerapps.md#faq). Vi støtter for øyeblikket ikke registrering med en personlig konto.

**Sp.** Kan jeg slette det individuelle miljøet mitt?

**Sv:** Du kan ikke slette dette miljøet på egen hånd. Leieradministratoren har tillatelse til å slette miljøet.
