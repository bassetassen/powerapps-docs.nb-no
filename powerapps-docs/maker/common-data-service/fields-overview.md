# <a name="fields-overview"></a>Feltoversikt

Feltene definerer de individuelle dataelementene som kan brukes til å lagre informasjon i en enhet. Opprett nye felter for å fange opp data når eksisterende standardenheter ikke har felter som oppfyller kravene dine. Etter at du oppretter et nytt felt, må du inkludere det i aktuelle skjemaer og visninger for enheten, slik at de er tilgjengelige i appen.
